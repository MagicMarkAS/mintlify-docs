The Conversation tab is a crucial feature of your AI agent, providing invaluable insights into user interactions. This powerful tool allows you to review conversations, improve your agent's performance, and share insights with clients.

## Key Features

  <Card title="Real-Time Monitoring" icon="eye">
    Watch live conversations as they unfold between users and your AI agent.

  <Card title="Performance Analysis" icon="chart-line">

  </Card>

## Interface Overview
The Conversation tab is divided into three main sections:
<Tabs>
  <Tab title="Left Panel">
    <img src="/images/left_section_shots.png" alt="Left Panel" width="400" />
    Displays a list of all conversations, sortable by various criteria.
  </Tab>
  <Tab title="Central Panel">
    <img src="/images/middle_section_shots.png" alt="Central Panel" width="700" />
    Shows the selected conversation transcript and session.
  </Tab>
  <Tab title="Right Panel">
    <img src="/images/right_section_shots.png" alt="Right Panel" width="400" />
    Provides conversation details and management options.
  </Tab>
</Tabs>


### Left Panel: Conversation List

<AccordionGroup>
  <Accordion title="Sorting Options">
    - Recent/Oldest: Sort by conversation start time
    - More/Less: Sort by number of interactions
    - By Tags: Filter conversations using custom or predefined tags
  </Accordion>
  <Accordion title="Conversation Fields">
    - ID: Unique identifier for each user (auto-generated)
    - Interactions: Number of user messages in the current chat
    - Timestamp: Start time of the conversation
  </Accordion>
</AccordionGroup>

### Central Panel: Conversation Transcript

<Steps>
  <Step title="Session Selection">
    Choose from multiple sessions at the top of the panel.
  </Step>
  <Step title="Conversation Flow">
    Review the back-and-forth between user and AI agent.
  </Step>
  <Step title="Pin Session UI">
    Click the pin icon to keep the session UI visible while scrolling.
  </Step>
</Steps>
<Note>
  A new session starts each time a user initiates a conversation on your website.
</Note>
 ### Right Panel: Conversation Management
Transcript URL
Share specific conversations easily with a unique URL.
<Tip>
  If configured, URLs are hosted on your agency domain and will always be whitelabeled.
</Tip>
